Quantum Leap API

Die einfachste und günstigste Pay-as-you-go API für KI-Bilder. Keine Abos, keine komplizierte Abrechnung.

Erste Schritte (Getting Started)

Dieses Projekt bietet dir programmatischen Zugang zu KI-Bildgenerierung zu extrem niedrigen Kosten. Perfekt für Prototypen, kleine Projekte und Bots.

Wichtiger Hinweis: Der API-Server wird aktuell über ngrok betrieben. Die URL kann sich täglich ändern. Prüfe diese Seite für die aktuell gültige URL.
Aktuelle API-Basis-URL: https://pseudomiraculous-milena-radially.ngrok-free.dev

API-Dokumentation

Schritt 1: Registrieren & API-Key erhalten

Sende eine POST-Anfrage, um einen Account zu erstellen und deinen persönlichen API-Key zu erhalten. Diesen Key brauchst du für alle weiteren Schritte.

curl -X POST \
  https://pseudomiraculous-milena-radially.ngrok-free.dev/register \
  -H 'Content-Type: application/json' \
  -d '{
    "username": "mein_cooles_projektname"
  }'

Schritt 2: Guthaben aufladen

Wir nutzen Gumroad für eine einfache und sichere Bezahlung. Guthaben wird in Blöcken von 5€ verkauft.

  1. Gehe zu unserer sicheren Gumroad-Produktseite: Guthaben hier kaufen.
  2. Klicke auf "Ich will das!" und gehe zum Checkout.
  3. **WICHTIG:** Gib im Feld "Dein API Key" deinen persönlichen API-Key aus Schritt 1 ein.
  4. Schließe den Kauf ab. Dein Guthaben wird automatisch gutgeschrieben.

Schritt 3: Bild generieren

Sende eine POST-Anfrage an den /v1/images/generate Endpunkt. Vergiss nicht, deinen API-Key im "Authorization"-Header zu senden!

curl -X POST \
  https://pseudomiraculous-milena-radially.ngrok-free.dev/v1/images/generate \
  -H 'Content-Type: application/json' \
  -H 'Authorization: DEIN_PERSOENLICHER_API_KEY' \
  -d '{
    "prompt": "Ein süßer Panda isst Bambus"
  }'
